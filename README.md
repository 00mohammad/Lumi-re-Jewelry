## 📖 About

**Lumière Jewelry** is a modern luxury jewelry landing page designed to provide an elegant and premium shopping experience.

The project focuses on creating a visually appealing interface with responsive layouts, smooth interactions, semantic HTML, and maintainable front-end code.

---

## 🎨 Design

The interface was designed in **Figma** before development.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/00mohammad/Lumi-re-Jewelry.git
```

### 2. Navigate to the project folder

```bash
cd Lumi-re-Jewelry
```

### 3. Open the project

Open **index.html** directly in your browser, or use **Live Server** in Visual Studio Code.
